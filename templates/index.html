GET Generate BRAT Image

/api/brat?text=Your+Text+Here&background=#000000&color=#FFFFFF
text (required) - The text to display in the BRAT image
background (optional) - Background color in hex format (e.g., #000000)
color (optional) - Text color in hex format (e.g., #FFFFFF)
Example:
/api/brat?text=Hello+World&background=%23000000&color=%23FFFFFF

GET Generate BRAT Video

/api/bratvid?text=Your+Text+Here&background=#000000&color=#FFFFFF
text (required) - The text to animate (words separated by spaces)
background (optional) - Background color in hex format
color (optional) - Text color in hex format
Example:
/api/bratvid?text=This+is+awesome&background=%23000000&color=%23FF0000
